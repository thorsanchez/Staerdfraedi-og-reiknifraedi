# REI201G - Stærðfræði og reiknifræði (2023)

Verkefni frá grunnnámskeiði í Python forritun og reikniritum í HÍ.

## námskeið

**Python fundamentals**
- Gagnagerðir, föll og skipulagseiningar
- String manipulation og data validation
- File handling og data import

**Algorithms og data structures**
- Sorting algorithms
- Search algorithms
- Tímaflækjugreining

**Data analysis**
- NumPy fyrir töluleg útreikninga
- Pandas fyrir gagnavinnslu
- Tölfræðileg greining gagna

Inniheldur fyrirlestradæmi (f), skilaverkefni (s) og tímadæmi (t) frá námskeiðinu
